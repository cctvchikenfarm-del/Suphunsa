# CKAP v3.4.0 — Multi-sheet Excel Import

## ก่อน Deploy

รัน `database/P3_MULTI_SHEET_IMPORT.sql` ใน Supabase SQL Editor หลัง migration P1 และ P2

## การใช้งาน

1. เปิด **เครื่องมือพิเศษ → นำเข้าข้อมูลรวม**
2. เลือกไฟล์ `.xlsx`
3. ระบบอ่านแท็บ ขยะ RDF, ทิชชู่, ถุงขยะ และอาหารหมา
4. ตรวจ Preview จำนวนรายการ ประเภท วันที่ หน่วย และสถานะซ้ำ
5. กด **นำเข้าครั้งเดียว**
6. หากเลือกไฟล์ผิด ใช้ปุ่ม **ย้อนกลับ** ที่ประวัติชุดนำเข้า

## กฎสำคัญ

- ค่า `0` เป็นข้อมูลจริงและถูกนำเข้า; ช่องว่างถูกข้าม
- ปี พ.ศ. และชื่อเดือนไทยถูกแปลงเป็น ค.ศ.
- `พศจิกายน` ถูกยอมรับเป็นพฤศจิกายน
- ตรวจข้อมูลซ้ำทั้งจากชุดนำเข้าเดิมและรายการที่คีย์ด้วยมือ
- ใช้รหัสอาหารหมามาตรฐาน `DOG_FOOD`
- ไฟล์ตัวอย่างของผู้ใช้ผ่าน 987/987 รายการ
